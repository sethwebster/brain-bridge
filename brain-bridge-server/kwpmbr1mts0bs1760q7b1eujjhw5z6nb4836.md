Built-in React APIs[](#undefined "Link for this heading")
=========================================================

In addition to [Hooks](../react.html) and [Components](components.html), the `react` package exports a few other APIs that are useful for defining components. This page lists all the remaining modern React APIs.

* * *

*   [`createContext`](createContext.html) lets you define and provide context to the child components. Used with [`useContext`.](useContext.html)
*   [`forwardRef`](forwardRef.html) lets your component expose a DOM node as a ref to the parent. Used with [`useRef`.](useRef.html)
*   [`lazy`](lazy.html) lets you defer loading a componentâ€™s code until itâ€™s rendered for the first time.
*   [`memo`](memo.html) lets your component skip re-renders with same props. Used with [`useMemo`](useMemo.html) and [`useCallback`.](useCallback.html)
*   [`startTransition`](startTransition.html) lets you mark a state update as non-urgent. Similar to [`useTransition`.](useTransition.html)
